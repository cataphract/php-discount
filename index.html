php-discount

PHP extension that wraps the discount Markdown library.
Compatible with PHP 5.2, 5.3 and trunk.

Documentation

The documentation is available here.

Download

Development version

You can download the source code of latest development version in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/cataphract/php-discount

Installation

Except for Windows, the procedure is to download and unpack the source code and, in the directory to where the files were unpacked, run: 

phpize && ./configure --with-php-config=/path/to/php-config && \
make && make test && sudo make install

For release download and installation can be handled by pecl by executing the giving commands.

The configuration file php.ini may have to be edited to load the extension.
